# TradingView

Plataforma principal de charting y análisis técnico.

## Uso en VQL

Se usa como fuente primaria de datos y análisis visual. Los patrones A+ se identifican inicialmente en TradingView antes de ser automatizados.

## Plan gratuito vs Premium

El plan gratuito es suficiente para la operativa actual.
